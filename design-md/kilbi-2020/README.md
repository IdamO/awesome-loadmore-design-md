# Kilbi 2020 Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Kilbi 2020](http://kilbi.badbonn.ch/2020/fr/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/kilbi-2020). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

## Mechanics Snapshot

- World systems: Club Instrument, Fan Shrine
- Archetype: Club Instrument
- Inputs: tap, drag, press
- Mobile fallback: Collapse to one active control strip, one focal stage, and tap-to-trigger presets instead of a dense multi-panel control surface.

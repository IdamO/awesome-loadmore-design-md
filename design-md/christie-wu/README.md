# Christie Wu Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Christie Wu](https://christiewu.fun/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/christie-wu). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

## Mechanics Snapshot

- World systems: Cozy Desktop, Collage Core
- Archetype: Portfolio Artifact
- Inputs: scroll, tap
- Mobile fallback: Keep the asymmetry in rhythm and type, but simplify the layout into a single authored column with strong anchors.

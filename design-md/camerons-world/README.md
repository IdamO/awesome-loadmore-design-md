# Cameron's World Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Cameron's World](https://www.cameronsworld.net/) website, cross-referenced with [manual curation](https://www.cameronsworld.net/). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

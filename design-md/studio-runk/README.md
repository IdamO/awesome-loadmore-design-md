# Studio Runk Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Studio Runk](https://studiorunk.com/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/studio-runk). ## Mechanics Snapshot

- World systems: Collage Core, Playable Poster
- Archetype: Toy Loop Microgame
- Inputs: tap, drag
- Mobile fallback: Keep only tap and drag; remove precision mechanics, shorten the loop, and enlarge hit targets.

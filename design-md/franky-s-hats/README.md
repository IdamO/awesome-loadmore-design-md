# Franky's Hats Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Franky's Hats](https://www.frankys-hats.com/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/franky-s-hats). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

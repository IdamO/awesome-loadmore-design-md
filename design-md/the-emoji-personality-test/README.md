# The Emoji Personality Test Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [The Emoji Personality Test](https://emoji.elwyn.co/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/the-emoji-personality-test). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

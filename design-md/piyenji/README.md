# Piyenji Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Piyenji](https://piyenji.com/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/piyenji). This is not the official design system. ## Mechanics Snapshot

- World systems: Collage Core, Cozy Desktop
- Archetype: Collage Field
- Inputs: scroll, hover, tap
- Mobile fallback: Flatten the field into a guided scavenger feed or chapter stack while preserving overlap, stickers, and hyperlink energy.

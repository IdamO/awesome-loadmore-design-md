# A proposal for immersive video on the web Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [A proposal for immersive video on the web](http://video.gabeferreira.com/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/a-proposal-for-immersive-video-on-the-web). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

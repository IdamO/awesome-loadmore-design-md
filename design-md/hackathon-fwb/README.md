# Hackathon FWB Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Hackathon FWB](https://hackathon.fwb.help/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/hackathon-fwb). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

# Art&Code8 Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Art&Code8](https://yehwansong.github.io/artcode8/#side) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/art-code8).

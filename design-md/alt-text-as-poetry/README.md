# Alt Text as Poetry Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Alt Text as Poetry](https://alt-text-as-poetry.net/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/alt-text-as-poetry). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

# To See The Wave As A Sliding Door Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [To See The Wave As A Sliding Door](https://toseethewaveasaslidingdoor.com/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/to-see-the-wave-as-a-sliding-door). ## Mechanics Snapshot

- World systems: Playable Poster, Collage Core
- Archetype: Playable Poster
- Inputs: scroll, tap, hover
- Mobile fallback: Split the poster into two to four scenes, preserve one hero interaction, and drop hover-only secondary behavior.

# Simon Rogers Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Simon Rogers](https://www.simonrogers.info/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/simon-rogers). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

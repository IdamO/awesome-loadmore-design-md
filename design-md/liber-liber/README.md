# A Scroll of One's Own Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [A Scroll of One's Own](http://liberliber.net/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/liber-liber). ## Mechanics Snapshot

- World systems: Collage Core, Club Instrument
- Archetype: Scroll Scrub Instrument
- Inputs: scroll, tap
- Mobile fallback: Replace long scrub zones with a horizontal slider or stepped scenes while keeping audio opt-in.

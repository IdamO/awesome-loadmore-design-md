# Dictionary of Online Behavior Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Dictionary of Online Behavior](http://www.dictionaryofonlinebehavior.com/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/dictionary-of-online-behavior). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

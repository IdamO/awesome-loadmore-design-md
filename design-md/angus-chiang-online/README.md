# Angus Chiang Online Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Angus Chiang Online](http://anguschiang.com) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/angus-chiang-online). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

## Mechanics Snapshot

- World systems: Fan Shrine, Luxury Archive
- Archetype: Commerce Shrine Stage
- Inputs: scroll, tap, hover, drag
- Mobile fallback: Use a single-column collectible feed with sticky purchase rails and clear release-state chips instead of shrinking desktop grids.

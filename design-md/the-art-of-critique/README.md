# The Art of Critique Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [The Art of Critique](https://theartofcritique.rietveldacademie.nl/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/the-art-of-critique). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

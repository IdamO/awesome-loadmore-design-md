# My Smol Site Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [My Smol Site](https://mysmol.site/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/my-smol-site). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

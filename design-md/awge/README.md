# AWGE Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [AWGE](https://www.awge.com/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/awge). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

## Mechanics Snapshot

- World systems: Fan Shrine, Club Instrument
- Archetype: Toy Loop Microgame
- Inputs: tap, drag
- Mobile fallback: Keep only tap and drag; remove precision mechanics, shorten the loop, and enlarge hit targets.

# Gucci Dive Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Gucci Dive](https://guccidive.gucci.com) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/gucci-dive). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

## Mechanics Snapshot

- World systems: Fan Shrine, Luxury Archive
- Archetype: Toy Loop Microgame
- Inputs: tap, drag
- Mobile fallback: Keep only tap and drag; remove precision mechanics, shorten the loop, and enlarge hit targets.

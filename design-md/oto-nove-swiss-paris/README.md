# Oto Nove Swiss Paris Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Oto Nove Swiss Paris](https://www.otonoveswiss.com/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/oto-nove-swiss-paris). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

# Fruitful school Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Fruitful school](http://www.fruitful.school) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/fruitful-school). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

## Mechanics Snapshot

- World systems: Cozy Desktop, Collage Core
- Archetype: Desktop Shell World
- Inputs: drag, tap, click
- Mobile fallback: Translate the window system into a card stack, bottom drawer, or tabbed shell while preserving the desktop fiction in the chrome.

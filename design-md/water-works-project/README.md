# Water Works Project Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Water Works Project](https://waterworksproject.nl/en) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/water-works-project). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

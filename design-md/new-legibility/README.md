# New Legibility Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [New Legibility](https://thehtml.review/04/new-legibility/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/new-legibility). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

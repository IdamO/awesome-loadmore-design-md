# Be Gen 21 Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Be Gen 21](https://jakedowsmith.studio/b-gen-21/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/be-gen-21). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

## Mechanics Snapshot

- World systems: Cozy Desktop, Playable Poster
- Archetype: Retro Tool Emulator
- Inputs: tap, drag, type
- Mobile fallback: Turn dense panels into a one-tool-at-a-time inspector or wizard while preserving the visual chrome.

# The Lost Lionesses Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [The Lost Lionesses](https://nike.martine-rose.com/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/the-lost-lionesses). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

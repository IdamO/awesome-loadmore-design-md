# Bon Iver Viisualiizer Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Bon Iver Viisualiizer](https://boniver.withspotify.com) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/bon-iver-viisualiizer). ## Mechanics Snapshot

- World systems: Club Instrument, Fan Shrine
- Archetype: Club Instrument
- Inputs: tap, drag, press
- Mobile fallback: Collapse to one active control strip, one focal stage, and tap-to-trigger presets instead of a dense multi-panel control surface.

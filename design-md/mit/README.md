# MIT's Center for Advanced Visual Studies Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [MIT's Center for Advanced Visual Studies](http://act.mit.edu/cavs) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/mit). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

## Mechanics Snapshot

- World systems: Luxury Archive, Club Instrument
- Archetype: Spatial Exhibition World
- Inputs: scroll, drag, tap, hover
- Mobile fallback: Switch to a guided tour with swipeable viewpoints and one-tap hotspot cycling instead of free camera control.

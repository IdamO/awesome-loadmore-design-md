# N10.AS Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [N10.AS](https://n10.as/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/n10-as). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

# Harley Weir Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Harley Weir](http://harleyweir.com/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/harley-weir). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

# Felipe Pantone: Contactless Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Felipe Pantone: Contactless](https://contactless.felipepantone.com/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/felipe-pantone-contactless). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

# For ever ware Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [For ever ware](http://foreverware.life/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/for-ever-ware). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

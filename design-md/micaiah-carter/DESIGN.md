# Design System Inspiration of Micaiah Carter

## 1. Visual Theme & Atmosphere

Micaiah Carter reads as restrained minimalism, exhibition-like framing. The live capture resolves to a light-leaning system built around #fefefe, #eaeaea, and accent notes from #c5c5c5. Discover Micaiah Carter’s debut monograph 'What’s My Name' on a visually striking website, featuring an interactive 3D book and animated pages floating in the background.

Desktop and mobile stay aligned but not identical: the desktop capture emphasizes a looser scroll narrative, while mobile compresses the same language into tighter interaction zones. The site's type system centers Graphik Web, which becomes the fastest way to reproduce the feel.

Key Characteristics:
- Primary tone: bright surfaces with editorial contrast
- Core palette: #fefefe, #eaeaea, #c5c5c5, #b9b9b9
- Typography stack: Graphik Web
- Desktop posture: flowing document rhythm
- Mobile posture: single-column immersive scroll

## 4. Component Stylings

### Web
- Buttons tend toward rgb(230, 228, 225) backgrounds with rgb(0, 0, 0) text.
- Links inherit #c5c5c5 as the interaction signal.
- Border radius trends: 6px.
- Shadow language: none.

### Mobile
- Mobile preserves the same palette while reducing surface area and increasing gesture weight.
- Recreate the mobile feel with oversized tap targets, single-column pacing, and typography that keeps Graphik Web in control.

## 5. Layout Principles

- Use a free-flowing vertical canvas on desktop.
- Keep mobile single-column and immersive rather than dashboard-like.
- Let the main background color (#fefefe) carry the atmosphere instead of layering multiple competing surfaces.
- Preserve asymmetry when present - the archive tags (photography, arts&culture, minimal) imply the site is intentionally non-generic.

## 6. Depth & Elevation

- Primary depth cue: none.
- Radius cue: 6px.
- Contrast cue: dark text on light surfaces with accent interruptions.

## 7. Do's and Don'ts

### Do
- Use Graphik Web consistently for headlines and interface labels.
- Keep the palette anchored to #fefefe, #eaeaea, and #c5c5c5.
- Preserve the experimental posture signaled by the loadmo.re tags: photography, arts&culture, minimal.
- Build separate desktop and mobile compositions instead of pretending one layout can fake both.

### Don't
- Don't genericize the interface into a default SaaS landing page.
- Don't introduce rounded, pastel, or glassmorphism defaults unless the captured site already does.
- Don't replace the extracted font stack with Inter/Roboto/system as the main voice unless no custom stack loaded.

## 8. Responsive Behavior

- Desktop capture uses screenshots/desktop.jpg as the visual baseline.
- Mobile capture uses screenshots/mobile.jpg as the mobile baseline.
- Keep touch targets oversized on mobile and allow the background system to dominate the viewport.
- If the live site failed to capture, fall back to the archival poster on the loadmo.re post page before inventing missing behavior.

# Communal Radio Club Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Communal Radio Club](https://communalradio.club/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/communal-radio-club). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

## Mechanics Snapshot

- World systems: Club Instrument, Cozy Desktop
- Archetype: Club Instrument
- Inputs: tap, drag, press
- Mobile fallback: Collapse to one active control strip, one focal stage, and tap-to-trigger presets instead of a dense multi-panel control surface.

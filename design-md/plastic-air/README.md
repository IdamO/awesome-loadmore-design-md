# Plastic Air Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Plastic Air](https://artsexperiments.withgoogle.com/plasticair/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/plastic-air). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

# Below the Surface Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Below the Surface](https://belowthesurface.amsterdam) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/below-the-surface). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

## Mechanics Snapshot

- World systems: Luxury Archive
- Archetype: Editorial Archive Index
- Inputs: scroll, tap, filter
- Mobile fallback: Keep a single-column feed, bottom-sheet filters, a persistent current-section pill, and inline detail expansion.

# Control The Virus 03 Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Control The Virus 03](https://controlthevirus.art/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/control-the-virus-03). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

## Mechanics Snapshot

- World systems: Luxury Archive, Club Instrument
- Archetype: Spatial Exhibition World
- Inputs: scroll, drag, tap, hover
- Mobile fallback: Switch to a guided tour with swipeable viewpoints and one-tap hotspot cycling instead of free camera control.

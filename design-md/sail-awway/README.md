# Sail Awway Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Sail Awway](http://sailawway.com) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/sail-awway). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

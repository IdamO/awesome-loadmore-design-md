# Edgar Schmitz Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Edgar Schmitz](http://edgarschmitz.com/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/edgar-schmitz). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

## Mechanics Snapshot

- World systems: Playable Poster, Luxury Archive
- Archetype: Portfolio Artifact
- Inputs: scroll, tap, drag
- Mobile fallback: Keep the asymmetry in rhythm and type, but simplify the layout into a single authored column with strong anchors.

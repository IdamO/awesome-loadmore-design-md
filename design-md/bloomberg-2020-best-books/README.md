# Bloomberg 2020 Best Books Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Bloomberg 2020 Best Books](https://www.bloomberg.com/features/2020-best-books/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/bloomberg-2020-best-books). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

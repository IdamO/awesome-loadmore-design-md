# thisisneverthat FA22 Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [thisisneverthat FA22](https://fa22.thisisneverthat.com/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/thisisneverthat-teaser). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

## Mechanics Snapshot

- World systems: Playable Poster, Fan Shrine
- Archetype: Playable Poster
- Inputs: scroll, tap, hover
- Mobile fallback: Split the poster into two to four scenes, preserve one hero interaction, and drop hover-only secondary behavior.

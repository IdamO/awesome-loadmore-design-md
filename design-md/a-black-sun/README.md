# A Black Sun Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [A Black Sun](https://nicolastilly.fr/a-black-sun/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/a-black-sun). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

# Ling's Cars Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Ling's Cars](https://m.lingscars.com/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/ling-s-cars). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

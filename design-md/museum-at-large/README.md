# Museum At Large Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Museum At Large](https://museumatlarge.com) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/museum-at-large). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

# Everything Can Be Scanned Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Everything Can Be Scanned](https://ecbscanned.com/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/everything-can-be-scanned). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

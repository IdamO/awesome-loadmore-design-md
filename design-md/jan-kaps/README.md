# Jan Kaps Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Jan Kaps](https://jan-kaps.com/real-books) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/jan-kaps). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

## Mechanics Snapshot

- World systems: Luxury Archive, Fan Shrine
- Archetype: Editorial Archive Index
- Inputs: scroll, tap, filter, hover
- Mobile fallback: Keep a single-column feed, bottom-sheet filters, a persistent current-section pill, and inline detail expansion.

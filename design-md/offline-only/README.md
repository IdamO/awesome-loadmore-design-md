# Offline only Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Offline only](https://chris.bolin.co/offline/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/offline-only). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

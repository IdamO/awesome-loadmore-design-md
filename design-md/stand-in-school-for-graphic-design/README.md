# Stand in School for Graphic Design Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Stand in School for Graphic Design](https://estende.eka-gd-ma.ee/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/stand-in-school-for-graphic-design). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

# Colour Mapper Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Colour Mapper](https://colour-mapper.michaelnorman.au/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/colour-mapper). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

## Mechanics Snapshot

- World systems: Cozy Desktop, Luxury Archive
- Archetype: Retro Tool Emulator
- Inputs: tap, drag, type
- Mobile fallback: Turn dense panels into a one-tool-at-a-time inspector or wizard while preserving the visual chrome.

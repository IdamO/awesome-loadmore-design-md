# Consecutives Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Consecutives](https://consecutives.work.damonzucconi.com/?id=litany-for-the-blind&width=99&fps=30&font_size=16px&break=false&text_align=justify) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/consecutives). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

## Mechanics Snapshot

- World systems: Collage Core, Playable Poster
- Archetype: Collage Field
- Inputs: scroll, hover, tap
- Mobile fallback: Flatten the field into a guided scavenger feed or chapter stack while preserving overlap, stickers, and hyperlink energy.

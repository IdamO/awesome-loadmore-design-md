# Odunpazarı Modern Museum Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Odunpazarı Modern Museum](https://omm.art/en/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/odunpazari-modern-museum). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

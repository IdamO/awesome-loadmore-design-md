## Mechanics Snapshot

- World systems: Club Instrument, Collage Core
- Archetype: Scroll Scrub Instrument
- Inputs: scroll, tap
- Mobile fallback: Replace long scrub zones with a horizontal slider or stepped scenes while keeping audio opt-in.

# Apple Airpods Pro Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Apple Airpods Pro](https://www.apple.com/airpods-pro/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/apple-airpods-pro). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

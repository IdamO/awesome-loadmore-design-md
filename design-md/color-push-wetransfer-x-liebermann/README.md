# Color Push : Wetransfer x Liebermann Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Color Push : Wetransfer x Liebermann](https://colorpush.wetransfer.com/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/color-push-wetransfer-x-liebermann). ## Mechanics Snapshot

- World systems: Cozy Desktop, Collage Core
- Archetype: Retro Tool Emulator
- Inputs: tap, drag, type
- Mobile fallback: Turn dense panels into a one-tool-at-a-time inspector or wizard while preserving the visual chrome.

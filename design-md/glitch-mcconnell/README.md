# Glitch Mcconnell Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Glitch Mcconnell](https://glitchmcconnell.netlify.app/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/glitch-mcconnell). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

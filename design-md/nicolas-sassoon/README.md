# Nicolas Sassoon Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Nicolas Sassoon](https://nicolassassoon.com/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/nicolas-sassoon).

# Swiss Design Awards 2020 Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Swiss Design Awards 2020](https://2020.swissdesignawardsblog.ch/everywhere) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/swiss-design-awards-2020). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

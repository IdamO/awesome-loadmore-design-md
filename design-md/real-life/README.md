# Real Life Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Real Life](https://reallifemag.com) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/real-life). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

## Mechanics Snapshot

- World systems: Luxury Archive, Collage Core
- Archetype: Editorial Archive Index
- Inputs: scroll, tap, filter
- Mobile fallback: Keep a single-column feed, bottom-sheet filters, a persistent current-section pill, and inline detail expansion.

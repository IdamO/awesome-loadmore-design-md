# Bids for Survival Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Bids for Survival](https://bidsforsurvival.com/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/bids-for-survival). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

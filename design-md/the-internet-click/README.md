# The Internet Click Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [The Internet Click](https://the-internet.click/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/the-internet-click). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

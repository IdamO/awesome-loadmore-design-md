# Centrer for Planetary Pedagogies Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Centrer for Planetary Pedagogies](https://www.centerforplanetarypedagogies.org/?ref=siteinspire) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/centrer-for-planetary-pedagogies). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

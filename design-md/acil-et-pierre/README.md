# Acil & Pierre Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Acil & Pierre](https://www.aciletpierre.fr/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/acil-et-pierre). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

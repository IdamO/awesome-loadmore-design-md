# Cash by Cash App Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Cash by Cash App](http://cashbycashapp.com/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/cash-by-cash-app). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

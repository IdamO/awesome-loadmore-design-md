# Woven sound Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Woven sound](https://woven-sound.surge.sh/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/woven-and-surge). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

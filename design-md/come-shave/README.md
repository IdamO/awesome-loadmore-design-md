# Come Shave Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Come Shave](https://www.comeshave.com/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/come-shave). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

## Mechanics Snapshot

- World systems: Collage Core, Playable Poster
- Archetype: Collage Field
- Inputs: scroll, hover, tap
- Mobile fallback: Flatten the field into a guided scavenger feed or chapter stack while preserving overlap, stickers, and hyperlink energy.

# model und ruine Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [model und ruine](https://modell-und-ruine.werkleitz.de/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/model-und-ruine). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

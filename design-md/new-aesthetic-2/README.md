# New Aesthetic 2 Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [New Aesthetic 2](https://newaesthetic.xyz/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/new-aesthetic-2). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

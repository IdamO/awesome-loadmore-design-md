# Collection of Holiday Experiences Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Collection of Holiday Experiences](https://collectionofholidayexperiences.xyz/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/collection-of-holiday-experiences). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

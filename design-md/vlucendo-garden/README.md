# Vlucendo Garden Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Vlucendo Garden](https://2021.vlucendo.com/garden/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/vlucendo-garden). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

# Business Fonts Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Business Fonts](https://www.businessfonts.biz/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/business-fonts). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

## Mechanics Snapshot

- World systems: Playable Poster, Collage Core
- Archetype: Playable Poster
- Inputs: scroll, tap, hover
- Mobile fallback: Split the poster into two to four scenes, preserve one hero interaction, and drop hover-only secondary behavior.

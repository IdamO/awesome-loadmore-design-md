# Feline Align Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Feline Align](https://mat-hill.xyz/feline_align) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/feline-align). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

## Mechanics Snapshot

- World systems: Luxury Archive, Fan Shrine
- Archetype: Spatial Exhibition World
- Inputs: scroll, drag, tap, hover
- Mobile fallback: Switch to a guided tour with swipeable viewpoints and one-tap hotspot cycling instead of free camera control.

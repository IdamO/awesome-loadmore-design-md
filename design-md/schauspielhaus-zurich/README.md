# Schauspielhaus Zürich Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Schauspielhaus Zürich](https://2122.schauspielhaus.ch/en/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/schauspielhaus-zurich). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

# 100k Studio Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [100k Studio](https://100k.studio/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/100k-studio). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

## Mechanics Snapshot

- World systems: Fan Shrine, Luxury Archive
- Archetype: Portfolio Artifact
- Inputs: scroll, tap
- Mobile fallback: Keep the asymmetry in rhythm and type, but simplify the layout into a single authored column with strong anchors.

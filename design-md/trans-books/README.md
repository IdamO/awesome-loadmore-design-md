# Trans Books Inspired Design System

[DESIGN.md](./DESIGN.md) extracted from the public [Trans Books](https://transbooks.center/2018/) website, cross-referenced with [loadmo.re](https://loadmo.re/posts/trans-books). This is not the official design system. The goal is to give an AI agent enough grounded design language to recreate the feel without flattening it into generic SaaS UI.

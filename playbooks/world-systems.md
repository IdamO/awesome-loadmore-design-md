# World Systems

Use these as repeatable design worlds. Each world includes a metaphor, scene logic, references in this repo, motion direction, and failure modes.

## 1. Club Instrument

Best for: music tools, artist products, launch pages that should feel like software made by people who actually love sound.

- Metaphor: sequencer, control surface, instrument panel, reactive booth
- Refs: `jazz-keys`, `blob-opera`, `holodec-world`, `intimacy-machine`, `desktop-fm`, `yaeji`
- Scene primitives: `HeroStage`, `LiveMeter`, `DropClock`, `ArtifactRail`
- Motion: pulsing loops, tempo-linked reveals, meter fills, subtle reactive glow
- Materials: dark chrome, signal green, waveform lines, display numerals, LEDs, blurred color bloom
- Type posture: elegant display serif + technical grotesk or mono
- Failure mode: turning into gamer UI or fake DAW cosplay
- Prompt seed: Build this like a release instrument, not a dashboard. The user should feel like they are touching a system with tempo.

## 2. Luxury Archive

Best for: culture tech, editorial products, fashion campaigns, anything that needs taste and authority without becoming cold.

- Metaphor: reading room, accession index, catalog object, museum label
- Refs: `harley-weir`, `developments`, `animals-as-objects`, `infringe`, `publication-studio`, `williams-college-museum-of-art`
- Scene primitives: `ArchiveIndex`, `StoryFrame`, `RoomNav`, `ScrapNote`
- Motion: restrained fades, image breathing, smart crop shifts, careful reveal timing
- Materials: warm paper, labels, captioning, object numbers, rules, grids, negative space
- Type posture: serif authority + clean sans workhorse + restrained mono metadata
- Failure mode: becoming sterile editorial minimalism with no pulse
- Prompt seed: Make it feel curated and expensive, but still digitally alive.

## 3. Collage Core

Best for: campaign pages, scene-y launches, youth culture moments, anything that should feel assembled from artifacts rather than components.

- Metaphor: poster wall, xerox stack, torn flyer, mixed media board
- Refs: `camerons-world`, `hybrid-vigor`, `finger-walking`, `glossary-of-undisciplined-design`, `hallo-internet`, `make-word-art`
- Scene primitives: `ScrapNote`, `TickerBand`, `ArtifactRail`, `CapsuleTile`
- Motion: layered reveals, offset scroll, jitter, drag, parallax scraps, deliberate overlap
- Materials: scans, torn masks, scribbles, tape, stickers, marks, color clashes, halftones
- Type posture: one disciplined base family plus one freak display face used sparingly
- Failure mode: pure mess with no compositional hierarchy
- Prompt seed: Build like a collage editor with a strong eye, not a random sticker explosion.

## 4. Cozy Desktop

Best for: community tools, soft productivity, playful archives, internet-native utilities, fan spaces.

- Metaphor: bedroom desktop, notes board, draggable windows, saved files, digital diary
- Refs: `simone-s-computer`, `hallo-internet`, `communal-radio-club`, `glossary-of-undisciplined-design`, `orb-farm`
- Scene primitives: `RoomNav`, `ArchiveIndex`, `ScrapNote`, `ArtifactRail`
- Motion: drags, snap-back, window shuffle, subtle bounce, hover wake-up
- Materials: paper notes, desktop chrome, status bars, folder labels, low-fi icons
- Type posture: humane sans + bitmap/mono utility face + occasional playful display accent
- Failure mode: gimmicky fake OS parody that gets in the way of reading
- Prompt seed: Keep it warm, navigable, and collectible. It should feel lived in, not novelty desktop cosplay.

## 5. Playable Poster

Best for: launches, single-purpose products, events, drops, and pages that should feel immediate and theatrical.

- Metaphor: interactive poster, moving flyer, one-stage campaign object
- Refs: `design-your-own-hole`, `gt-flexa`, `gt-maru-typeface`, `colour-mapper`, `circuits-of-truth`, `smc-real-time`
- Scene primitives: `HeroStage`, `TickerBand`, `DropClock`, `StoryFrame`
- Motion: big reveals, draw-on SVGs, poster-scale transitions, hover transforms with conviction
- Materials: huge type, sparse copy, graphic marks, lines, badges, loud field colors
- Type posture: display-first, with a stable supporting sans beneath it
- Failure mode: becoming a static poster with no interaction depth
- Prompt seed: Treat the page itself as the campaign artifact.

## 6. Fan Shrine

Best for: artist pages, merch worlds, edition drops, community products, fandom-led commerce.

- Metaphor: shrine, merch table, collectible shelf, devotion board
- Refs: `online-ceramics`, `communal-radio-club`, `country-music`, `the-wisdom-of-the-third-wave`, `issued-by-bottega`, `martine-rose`
- Scene primitives: `CapsuleTile`, `ArtifactRail`, `StoryFrame`, `DropClock`
- Motion: collectible flips, hover enlargement, ambient drift, reveal by edition or release status
- Materials: tags, edition labels, stamps, product cards, notes, relic photography
- Type posture: emotionally loud display + grounded product UI face
- Failure mode: turning into basic e-commerce with nicer branding
- Prompt seed: Make buying feel like collecting a relic from a scene, not checking out from a store.

## Selection rule

Choose one dominant world.
Borrow one secondary interaction cue from another world.
Borrow one typography cue from a third.
Do not blend more than three worlds into a single page.
